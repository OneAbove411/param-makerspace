User Roles — Access Matrix
ROLE
Viewer
Read-only access. Browse everything, can't submit or interact. Pre-registration state.
ROLE
Maker
Standard user. Submit projects, complete challenges, earn XP, attend events, access store.
ROLE
Mentor
Review submissions, provide feedback, create challenges, guide makers. Requires Maker rank+.
ROLE
Admin
Full platform control. Manage users, approve projects, create events, configure gamification, moderate content.
Critical User Journeys
JOURNEY 1 — FIRST-TIME USER
Home → "See How It Works" ↓
→ Maker Loop (Pick Quest → Build → Level Up)
→ "Create Your Account" →
→ Register → Profile Setup Wizard
→ Dashboard (first quest suggested)
→ Explorer Hub → First Challenge
→ Submit → Earn XP → Rank to Tinkerer
JOURNEY 2 — RETURNING MAKER
Login → Dashboard
→ "Next Quest" card → Challenge Detail
→ Complete + Submit work
→ XP awarded → Check rank threshold
→ Continue or Rank Up celebration
→ Back to Dashboard (updated XP bar)
→ Browse Projects / Check Store unlocks
JOURNEY 3 — EXPLORATION LOOP
Dashboard → Makers Directory
→ Browse maker profiles by domain
→ View a maker's projects
→ Get inspired → Start own project
→ Dashboard → "Propose a Project"
→ Project submitted → XP + Badge check
→ Badge Catalog (see new unlocks)
